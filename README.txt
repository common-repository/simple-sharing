Adds a sharing tool to every page and post.

== Description ==
Tired of sharing plugins that route all your shares through their site, or weigh your site down with massive bloatware?

We were, so we made this simple sharing plugin that just adds a few links for your visitors to share posts and pages directly to the selected social media hassle-free.

By using our shortcode `[exxica_sharing]` you can add the sharing tool to any page you want.

= Features =
- Does not route through our servers or any other servers.
- Does not require any external libraries.
- A lightweight plugin that only does what it's supposed to do.
- Requires no extra installation. Works out of the "box".
- 2 icon sets to choose from.
- A multitude of options to add some variety to your sharing tools.
- Available in: English, Norwegian (Bokmal)

== Frequently Asked Questions ==
= Where can I find the plugin after it's installed? =
The plugin adds a shortcode `[exxica_sharing]` to your WordPress. You can use this in your text to show the sharing tool.

== Installation ==
= Automatic installation =
1. Press **Install** to install this plugin to your WordPress.
2. Enable the plugin in WP admin.

= Manual installation =
1. Download `simple-sharing.zip` from any available source.
2. Connect to your FTP server and upload and extract the zip-archive to your `/wp-content/plugins/` folder.
3. Enable the plugin in WP admin.
